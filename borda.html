Resumo

Border radius generator foi um projeto criado com o objetivo de ajudar a criar o código de estilo para CSS com compatibilidade.

Descrição

Este projeto possui 4 entradas de dados os quais vão definir o tamanho do raio da borda em pixels, o usuário pode ver o resultado no mesmo momento. Ao digitar algum valor é gerado um código para CSS com a propriedade border-radius com compatibilidade para a maioria dos navegadores (-webkit-border-radius, moz-border-radius e border-radius). O usuário pode tanto selecionar o texto e copiar como clicar num botão que copia para sua area de transferência através do JavaScript.

Tecnologias

Foi utilizado o HTML5 para marcação, CSS3 para formatação do estilo e JavaScript sem frameworks para a interação do site.

JavaScript

Foi criado um função copia() para fazer a cópia do texto dentro area de texto. Para a mudança da propriedade border-radius foi criado a função raio() que formata a caixa branca e caso os 4 valores sejam o mesmo sera criado a propriedade somente com um valor, para isto serão chamadas as funções atuliza1() e atuliza4().
