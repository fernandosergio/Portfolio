Resumo

Este site faz a função de um relógio de Pomodoro.

Descrição

Tem dois contadores, um para o tempo de trabalho e outro para o tempo de descanso. Ao clicar em play ele inicia o contador do tempo de Trabalho, quando o contador chegar em 00:00 irá ficar cinza e dispara um som de alerta. Então o o contador de Descanso inicia a contagem e muda sua cor para verde, quando chegar a 00:00 dispara um som de alerta, muda sua cor para cinza e a contagem em Trabalho se incia novamente, assim formando um loop.

Tecnologias

Foi utilizado o HTML5 para marcação, CSS3 para formatação de estilo e JavaScript sem frameworks para a interação do site.

JavaScript

As funções criadas foram conta(), para fazer a contagem do temporizador de Trabalho; contades(), para a contagem do temporizador de Descanso; chama(), para iniciar o cronometro e verificar qual estava ativo, sendo o de Trabalho o padrão; para(), para pausar o cronometro; reinicia(), para reiniciar o cronometro.
