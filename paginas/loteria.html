Resumo

Esse site faz uma comparação dos números apostados com o resultado oficial da loteria.

Descrição

O usuário deve escolher o concurso desejado, ao escolher será criado dinamicamente a quantidade de números de acordo com cada concurso, deve digitar o número do concurso e selecionar os números que deseja comparar. Ao clicar no botão será feito uma requisição à API da loteria, será comparado com o arquivo consumido e mostrará os números acertados e errados e o total de acertos.

Tecnologias

Foi utilizado HTML5 para marcação, CSS3 para formatação de estilo e JavaScript com o framework JQuery para a interação do site.

JavaScript

Ao selecionar algum concurso será criado varios números dinamicamente, que são input type="checkbox", caso o número do concurso não for inserido será recebido o último concurso de acordo com a API da loteria. Para não ter problemas com a quantidade de números selecionados é feito uma verificação com o tipo do concurso e a quantidade de inputs selecionados. Então, ao clicar no botão será feito uma requisição do tipo GET com .ajax, no caso de sucesso irá comparar e mostrar na tela o resultado.
