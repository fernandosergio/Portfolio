Resumo

Este é um clone do site google glass, no qual iniciei o estudo sobre tecnologias.

Descrição

Este site possui uma aba home, que possui informações sobre o google glass e videos mostrando o funcionamento; aba especificações, aonde o usuário pode interagir com algumas partes do equipamento google glass que mostra algumas especificações do produto; aba fotos e multimídia, mostram fotos, videos e audio sobre o produto; e a aba fale conosco, onde é possível criar uma conta, com nome senha, email, gênero, data e endereço, também é possível mandar uma mensagem e adquirir um produto.

Tecnologias

Foi utilizado o HTML5 para marcação, CSS3 para formatação do estilo e JavaScript sem frameworks para a interação do site.

JavaScript

Foi utilizado JavaScript apenas para mudar a foto ao selecionar os menus.
